- **tailwind** — Tailwind CSS integration for modern responsive styling.

## ▶️ How to Run a Project

Each project is self-contained. To run any project:

1. Open a terminal in the project folder, for example `card` or `router`.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the local URL shown in the console.
